# educational-rover-hardware

教育用ローバーのハードウェア仕様

オンボードコンピューターにRaspberry Pi Zero/Wを使用した教育用ローバーの設計書です。

## ピン配置

### DRV8835

| Part/Leg | Raspberry Pi GPIO |
| --- | --- |
| AIN1 | 18 |
| AIN2 | 23 |
| BIN1 | 24 |
| BIN2 | 25 |

### LEDs

| Part/Leg | Raspberry Pi GPIO |
| --- | --- |
| LED1 | 16 |
| LED2 | 20 |
| LED3 | 21 |
